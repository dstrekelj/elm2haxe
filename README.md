# elm2haxe

Implementation of [the Elm Language
examples](http://elm-lang.org/Examples.elm) in Haxe.

## Running Examples

Examples require [Haxe 3.2.0](http://haxe.org/download/).

```
make [exercise name]
```

example:

```
make js.image
```
